# Omega Contracting Website

A professional, pixel-perfect website for Omega Contracting - a licensed general contractor serving Metro Atlanta.

## Overview

This website showcases Omega Contracting's construction services, project portfolio, and provides an easy way for potential clients to request estimates.

## Features

- **Homepage**: Hero section with compelling CTA, services overview, recent projects gallery, and client testimonials
- **Services Page**: Detailed service offerings with pricing and feature breakdowns
- **Projects Page**: Portfolio gallery with case studies and detailed project information
- **Contact Page**: Comprehensive contact form for estimate requests with budget range selection

## Design System

The website follows a professional construction industry design aesthetic:

- **Primary Color**: Navy Blue (#2C5F8D) - Headers, sections, trust elements
- **Accent Color**: Orange (#F68B1F) - CTAs, icons, emphasis
- **Typography**: Inter font family - Clean, modern sans-serif
- **Layout**: Container max-width 1200px, responsive grid systems
- **Components**: Sharp corners on cards, rounded buttons (4px), orange circular icon backgrounds

## Technology Stack

- **Framework**: Next.js 15.5.2 with App Router
- **Styling**: Tailwind CSS 4.0 with custom HSL color system
- **Icons**: Lucide React
- **Images**: AI-generated professional construction photography
- **Type Safety**: TypeScript

## Pages

1. **Home** (`/`) - Landing page with hero, services, projects, testimonials
2. **Services** (`/services`) - Detailed service listings with pricing
3. **Projects** (`/projects`) - Portfolio gallery with case studies
4. **Contact** (`/contact`) - Estimate request form with contact information

## Running Locally

```bash
npm install
npm run dev
```

Visit `http://localhost:4006`

## Build for Production

```bash
npm run build
npm start
```

## Key Components

- `Header` - Sticky navigation with logo, menu, phone number, and CTA
- `Footer` - Contact info, quick links, social media, business hours
- `ServiceCard` - Service display with icon, title, description
- `ProjectCard` - Project showcase with hover overlay effects
- `TestimonialCard` - Client testimonial with 5-star rating
